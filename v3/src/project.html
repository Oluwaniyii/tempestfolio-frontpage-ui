Tempestfolio

node js | 5-11-2022

Tempestfolio is a  portfolio website with admin control room that makes it easier to update projects and content without having to revisit the code base

 

Languages and tools

  • HTML & CSS
  • Vanilla Javascript
  • Ajax
  • Node JS
  • Express
  • Heroku

 

Features

  • Responsive soft User Interface
  • Admin control room authentication
  • Managing and storing dynamic data to a cloud database
  • Uploading and management of resume files to a third party data resource
  • Receiving mails from the Frontpage

 

Future plans

The API it consumes is tightly coupled to the app, Later in the future I wish to decouple the rest API to an independent API for easier switch in front-end tech and add more flexible endpoints

Addition of real time statistics on number of visitors, CV downloads or even media views on YouTube, IP tracking if not creepy

Switching to one page applications on the client

 

Drawbacks

The APP pages are genarated on the server which  kind of makes the API rigid and hard to make flexible updates on features like project media  or extend the API easily which is bad

What I plan to do is switch to an independent stand-alone API, this way its easier to extend and the front-end doesn't have to define what the API should look like, rather the API will have vast options for front-tend to choose from.

 

Pride

Whats lovable about this application should be  the UI, plus the fact that changes can be made with little or no interaction with the code
